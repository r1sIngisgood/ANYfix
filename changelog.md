## [Update] - 2026-02-02 - v1.4.4

### Added
- **Announce Feature**: Добавлена возможность настройки текста объявления ("Announce"). Текст (поддерживается многострочный ввод) отображается в клиентах Happ/Karing в виде информационного баннера.
- **Admin Panel**: В раздел "Configure Link" добавлено поле для редактирования текста объявления (Announce Text).

### Fixed
- **Happ Import**: Исправлен импорт подписки в клиенты Happ/Karing. Реализован специальный маршрут для принудительной отдачи текстового конфига, обходящий проверки User-Agent.
- **Info Button**: Исправлена работа кнопки "Информация" (Info) в клиентах. Теперь она корректно открывает веб-панель управления подпиской пользователя, а не Support URL.
- **Headers Encoding**: Реализовано корректное Base64-кодирование заголовков объявления для поддержки спецсимволов и кириллицы.
