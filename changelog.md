## [Update] - 2026-01-31 - v1.4.2

### Fixed
- **Backup & Restore**: Исправлена критическая ошибка, из-за которой кнопка "Restore" не реагировала на выбор файла. Добавлен индикатор прогресса восстановления.
- **Telegram 2FA**: Исправлен баг "Exit code -15" при включении двухфакторной аутентификации. Теперь перезагрузка панели происходит корректно без разрыва соединения.
- **Service Stability**: Устранены причины падения панели при перезагрузке службы через веб-интерфейс. Скрипт управления теперь корректно обрабатывает перезапуск процесса.
- **Config Repair**: Добавлена автоматическая коррекция "слипшихся" строк в файле конфигурации `.env`.

### Changed
- **System**: Обновлена логика скрипта восстановления бэкапов для повышения надежности.
- **UI**: Добавлена анимация загрузки при восстановлении из бэкапа.
